Information about the tags we have used so far:

Tag Attributes Comments
h1-h6 Heading, levels 1 to 6
p Paragraph
ol Ordered list
ul Unordered list
li List item
strong Makes text items stand out by making them bolder
em Makes text items stand out by making them italic
img src, alt, width, height Displays an image
a href, target, title Turns the enclosed content into an anchor, i.e. a link that the user can click on
table Display data as a table
thead The head section of a table
tbody The body section of a table
tr A table row
th A table header cell
td A table data cell
