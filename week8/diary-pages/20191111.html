Monday November 11th 2019

We learned about the various attributes that can be used on form tags:

  • action - where to send the data from the form (URL)
  • method - how to send the data from the form (GET/POST)
  • autocomplete - whether the browser should use autocomplete (on)
  • name - the name of the form, used if there is more than one form
  • id/class - used for styling or JavaScript element targeting

We learned about getting data from the user via HTML forms, using various types of input fields:

  • <input type="text">
  • <input type="submit">
  • <input type="password">
  • <input type="checkbox">
  • <input type="radio">
  • <select>
  • <textarea>
  • <input type="hidden">
