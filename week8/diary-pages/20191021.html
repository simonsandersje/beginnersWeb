Monday October 21st 2019

In the third lesson, we learned that a website was a collection of webpages.

We learned about saving files into a folder structure and linking between files using HTML anchor tags.

We also learned about the differences between:

  • External links
  • Absolute links
  • Relative links

We used this knowledge to move various parts of our code to different folders/files and to link between those files to turn our single webpage into a small website.
