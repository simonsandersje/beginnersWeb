Monday October 14th 2019

In the second week, we learned that semantic HTML can help give the user contextual information about the content.

We learned about the following tags:

  • h1 - heading (level 1)
  • p - paragraph
  • h2-h6 - headings (levels 2-6)
  • ol - order list (e.g. numbered)
  • ul - unordered list (e.g. bulleted)
  • li - list item within a list
  • - an important section of text
  • em - a section of text that should be emphasised
  • a - an anchor tag that links the user to other content
  • img - an image

We learned how to take a snapshot of the computer screen and save it as an image file.

We were introduced to VSCode - the code editing software used by many developers.
